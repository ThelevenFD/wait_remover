# Wait Remover 插件

一个简单的 MaiBot 插件，用于移除聊天中的 wait 动作，让对话更加流畅。

## 功能

- 移除私聊中的 wait 动作
- 让聊天机器人更积极地参与对话
- 避免不必要的等待时间

## 安装

1. 将插件文件夹复制到 `plugins` 目录
2. 重启 MaiBot

## 配置

在 `config.toml` 中可以配置：

```toml
[plugin]
config_version = "1.0.1"
remove_wait_action = true  # 是否启用移除 wait 动作
```

## 使用

插件启动后会自动生效，无需额外操作。
